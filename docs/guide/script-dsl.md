# Script DSL

vnova-engine supports the following step types:

- label
- scene
- image
- show
- hide
- stop
- effect
- say
- think
- narrate
- choice
- input
- select
- modal
- jump
- bgm
- sfx
- video
- particles
- notify
- wait
- call
- end

See the full documentation in the README for detailed usage examples.
